##  P2P Video Chat With Node.Js

# 👨🏻‍🔧 What
Node.Js와 Socket.Io의 candidate를 이용한 P2P 화상 채팅 웹 어플리케이션입니다.

#  🚀 How

```js
$npm install
$npm run dev
```

방 이름 입력 후 카메라 및 마이크 권한 허용해야 기능을 실행할 수 있습니다.
